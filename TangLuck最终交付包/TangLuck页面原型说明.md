# Tang Luck 页面原型说明

## 0. 高保真原型入口

已补充可直接打开的高保真 HTML 原型：

| 文件 | 用途 |
| --- | --- |
| `TangLuck高保真原型.html` | 用于客户评审、产品/运营确认、前端页面拆分、QA 对照验收 |

原型包含两套视图：

| 视图 | 覆盖页面 |
| --- | --- |
| C 端用户 App | 首页、钱包、活动中心、Gold Coins 商店、兑换、客服与 AMOE |
| 后台运营管理 | BI 看板、用户管理、钱包流水、活动配置、商品包配置、兑换审核、AMOE 审核、客服工单、审计日志 |

原型中的按钮、状态、弹窗用于表达业务流程和验收口径，不代表最终前端组件实现方式。

## 1. C 端页面

| 页面 | P0-A 元素 | P1/P2 增强 |
| --- | --- | --- |
| 注册页 | 邮箱、密码、年龄确认、Terms/Rules/Privacy 勾选 | 手机号、验证码、重签 |
| 首页 | 资产、欢迎奖励、每日登录、任务、Coupon、合规页脚 | 分层推荐、会员 |
| 钱包页 | GC、SC、冻结 SC、流水、SC 来源说明 | 兑换状态、筛选 |
| 活动中心 | 注册赠送、每日登录、每日任务、Coupon | 邀请、排行榜、转盘 |
| 商店页 | 灰态/预留 | GC 商品包、订单状态 |
| 兑换页 | 灰态说明 | KYC、申请、状态、申诉 |
| AMOE 页 | 静态说明 | 表单、状态查询 |
| 客服页 | 工单入口、FAQ | 业务单自动关联 |

## 2. 后台页面

| 页面 | 主要能力 |
| --- | --- |
| 用户管理 | 用户资料、地区、风险等级、钱包摘要 |
| 钱包流水 | 查询 GC/SC 变化、来源、幂等 key |
| 活动配置 | 类型、地区、奖励、预算、SC 策略、规则版本 |
| 商品包配置 | GC 数量、价格、SC bonus、地区、支付渠道 |
| 兑换审核 | KYC、冻结 SC、风险、审核动作 |
| AMOE 审核 | 申请资料、频率、地区、发放 SC |
| 客服工单 | 工单分类、关联业务、补偿审批 |
| BI 看板 | 注册、领取、SC、购买、兑换、投诉、风控 |
| 审计日志 | 操作人、对象、前后值、时间、IP |
