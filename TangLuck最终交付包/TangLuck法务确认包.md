# Tang Luck 法务确认包

## 1. 确认目标

用于提交美国律师确认 Tang Luck 的 sweepstakes 产品模式、州级开放、SC 发放、AMOE、GC 商品包、兑换规则和 App/推广文案。

## 2. 必须确认项

| 编号 | 事项 |
| --- | --- |
| L-001 | 开放州、禁止州、审核州 |
| L-002 | Terms、Sweepstakes Rules、Privacy、AMOE 文本 |
| L-003 | No Purchase Necessary 入口和 AMOE 申请方式 |
| L-004 | 注册赠送、每日登录、每日任务是否可发小额 SC |
| L-005 | Coupon、邀请、排行榜、转盘、会员是否可发 SC |
| L-006 | GC 商品包是否允许 SC bonus |
| L-007 | 兑换门槛、KYC、打款、SLA 表述 |
| L-008 | NY/FL 等州 prize pool 阈值、注册、保证金、winner list |
| L-009 | App 商店文案、广告文案、客服话术 |

## 3. 输出物

| 输出 | 用途 |
| --- | --- |
| 州级开放确认表 | 配置 `compliance_regions` |
| 规则版本确认 | 配置 `compliance_documents` |
| SC 发放确认 | 配置 `sc_policy_rules` |
| 商品包确认 | 决定是否开启 SC bonus |
| 高风险事项清单 | 上线阻断或灰态处理 |
