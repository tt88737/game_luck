# Tang Luck App 上架审核包

## 1. 审核目标

用于准备 iOS / Android 提审材料。本文档不保证 Apple/Google 审核通过，审核结果以平台实际反馈为准。

## 2. 提审材料

| 材料 | iOS | Android |
| --- | --- | --- |
| 审核账号 | 必须 | 必须 |
| 产品说明 | 必须 | 必须 |
| Sweepstakes Rules | 必须 | 必须 |
| No Purchase Necessary / AMOE | 必须 | 必须 |
| 支付说明 | 必须 | 必须 |
| 地区限制说明 | 必须 | 必须 |
| 年龄分级说明 | 必须 | 必须 |
| Privacy Policy | 必须 | 必须 |
| Responsible Gaming | 建议 | 建议 |
| 审核问答 FAQ | 必须 | 必须 |

## 3. 审核 FAQ

| 问题 | 标准回答方向 |
| --- | --- |
| 是否销售可兑换币 | 不销售 SC，只销售 GC |
| 是否必须购买才能参与 | 否，提供 No Purchase Necessary / AMOE |
| 是否支持真钱博彩 | 产品定位为娱乐和促销 sweepstakes，具体表述由法务确认 |
| 哪些地区可用 | 按州级配置开放，未开放地区限制功能 |
| Apple/Google 是否参与活动 | 不参与、不赞助 |
